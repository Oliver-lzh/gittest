### 2.遇到问题

了解到回溯的解题的基本构造和总体框架中，第一次不知道代码上如何实现。



### 3.思路

和递归算法一样，需要三部曲。

1.确认函数的参数

2.确认返回条件

3.单层逻辑

其中回溯算法的算法框架一般如下：

```python
result = []
def backtrack(路径, 选择列表):
    if 满足结束条件:
        result.add(路径)
        return
    
    for 选择 in 选择列表:
        做选择
        backtrack(路径, 选择列表)
        撤销选择
```



### 4.题解

```c++
class Solution {
private:
    vector<int> path;
    vector<vector<int>> result;
    void backtrack(int n,int k,int indexstart)
    {
        if(path.size()==k)
        {
            result.push_back(path);
            return ;
        }

        for(int i=indexstart;i<=n;i++)
        {
            path.push_back(i);
            backtrack(n,k,i+1);
            path.pop_back();
        }
    }
public:
    vector<vector<int>> combine(int n, int k)
    {
        result.clear();
        path.clear();
        backtrack(n,k,1);
        return result;
    }
};
```
